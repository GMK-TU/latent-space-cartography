# spider-logo
Spiders to scrape the web for logo images

## Getting Started

1. Clone the repository

2. Ask @yyyliu for data and a database dump

3. Useful commands
```
cd client

# install dependencies
npm install

# transpile client-side code
npm run dev

# start the server
python server.py
```

4. Navigate to http://localhost:5000/#/ in your browser
